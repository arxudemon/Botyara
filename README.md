# Genshin Gacha Discord Bot

Фан-бот, симулирующий систему молитв (гачи) из Genshin Impact: pity-система,
soft/hard pity, 50/50, GIF-анимация раскрытия карточки.

## ⚠️ Важно про авторские права

Genshin Impact, имена персонажей, иконки, CG-анимации и звуки — собственность
**HoYoverse / miHoYo**. Этот проект — независимая фан-реализация:

- Все GIF-анимации генерируются **программно** (Pillow), без использования
  оригинальных изображений/видео из игры.
- В `data/*.json` используются только **имена** персонажей и оружия (факты,
  не защищены авторским правом сами по себе) — не изображения.
- Если вы захотите добавить портреты персонажей — используйте их только для
  личного/некоммерческого использования на своём приватном сервере. Публичный
  бот с полным набором оригинальных CG-анимаций и озвучки может нарушать ToS
  HoYoverse и авторские права. Публиковать бота в маркетплейсах ботов Discord
  с таким контентом не рекомендуется.

## Установка

```bash
git clone <ваш-репозиторий>
cd genshin-gacha-bot
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

Создайте файл `.env` в корне проекта:

```
DISCORD_TOKEN=ваш_токен_бота
```

Токен получаете в https://discord.com/developers/applications — создайте
приложение → Bot → Reset Token. В разделе OAuth2 → URL Generator включите
scope `bot` и `applications.commands`, права: Send Messages, Embed Links,
Attach Files, Use Slash Commands.

## Запуск

```bash
python3 bot.py
```

После первого запуска слэш-команды синхронизируются автоматически
(может занять до часа при глобальной синхронизации — для быстрой
разработки синхронизируйте команды на конкретном сервере через
`bot.tree.sync(guild=discord.Object(id=YOUR_GUILD_ID))`).

## Команды

| Команда | Описание |
|---|---|
| `/pull banner:<Персонажи/Оружие>` | Одна молитва |
| `/pull10 banner:<Персонажи/Оружие>` | 10 молитв подряд |
| `/inventory` | Список полученных предметов |
| `/pity` | Текущий прогресс pity и баланс |

## Как расширить список персонажей/оружия

В `data/characters.json` и `data/weapons.json` сейчас лежит ограниченная
выборка (~20 и ~13 записей). Чтобы получить полный актуальный список:

1. Используйте открытые API комьюнити, например Ambr (`https://api.ambr.top/v2/en/avatar`)
   или Enka.Network — они отдают JSON с именами, редкостью, элементом и типом оружия.
2. Напишите скрипт, который скачивает и раскладывает данные в формат,
   совпадающий с текущими JSON-файлами (`name`, `rarity`, `element`,
   `weapon_type`, `banner`).
3. Не забудьте различать `banner: "limited"` (баннерные, ивентовые) и
   `"standard"` (доступны всегда) — это критично для работы системы 50/50.

## Структура проекта

```
bot.py              — точка входа
config.py           — вероятности, pity, токен
gacha_system.py      — вероятностная логика (pity, 50/50)
database.py         — SQLite: pity и инвентарь пользователей
animator.py          — генерация GIF-анимации раскрытия карточки
cogs/gacha_cog.py    — слэш-команды Discord
data/characters.json — база персонажей (нужно дополнить)
data/weapons.json     — база оружия (нужно дополнить)
```
